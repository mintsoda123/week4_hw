## 1. 개요

### 1.1 제품 정의

Week 4 Physics Neural Network Dashboard는 물리 데이터를 Neural Network로 학습·예측하는 4가지 실험을 하나의 GUI 애플리케이션으로 통합한 학습용 대시보드입니다. 사용자는 하이퍼파라미터를 조절하고 학습 버튼 하나로 실시간 Loss 곡선과 예측 결과를 확인할 수 있습니다.

### 1.2 배경

기존에는 4개의 독립 스크립트(`01perfect1d.py` ~ `04pendulum.py`)를 각각 터미널에서 실행하고, 완료 후 PNG 파일을 열어 결과를 확인해야 했습니다. 이 방식은 파라미터 실험이 불편하고 학습 과정을 실시간으로 관찰할 수 없었습니다. 이를 해결하기 위해 PySide6 기반의 통합 대시보드를 개발합니다.

### 1.3 목표

- 4개 Lab 실험을 하나의 창에서 실행
- 하이퍼파라미터 실시간 조절 및 학습
- 학습 중 Loss 곡선과 예측 결과를 실시간 시각화
- 학습 중단 및 리셋 기능으로 반복 실험 지원

---

## 2. 사용자 및 사용 맥락

### 2.1 대상 사용자

- **주 대상:** AI 물리 과정 수강생 (Week 4 실습)
- **기술 수준:** Python 기초, TensorFlow 입문 수준
- **사용 환경:** Windows 로컬 머신 (Python 3.11, GPU 없는 CPU 환경)

### 2.2 사용 시나리오

1. 수강생이 `python app.py`로 대시보드를 실행한다.
2. 사이드바에서 실험하고 싶은 Lab을 선택한다.
3. Epochs, Learning Rate 등 하이퍼파라미터를 조절한다.
4. **▶ 학습 시작**을 눌러 학습을 시작하고, Loss 곡선과 예측 그래프가 실시간으로 갱신되는 것을 관찰한다.
5. 파라미터를 바꿔 **↺ 리셋** 후 재학습하며 결과를 비교한다.

## 3. 기능 요구사항

### 3.1 공통 기능

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| F-01 | 사이드바에서 Lab 1~4로 전환 | 필수 |

| F-03 | ▶ 학습 시작 버튼: 백그라운드 스레드에서 학습 실행, UI 비동기 갱신 | 필수 |
